# 🚀 TaskFlow - Landing Page SaaS

TaskFlow adalah landing page sederhana untuk produk SaaS (Software as a Service) yang berfokus pada manajemen tugas dan produktivitas.

## 🎯 Deskripsi
Landing page ini dibuat sebagai tugas Modul 6 dengan tujuan:
- Menerapkan semantic HTML5
- Menggunakan CSS untuk styling dan layout
- Membuat desain responsive
- Deploy ke GitHub Pages

## ✨ Fitur
- Hero section (headline + CTA)
- Features section (3 fitur utama)
- Testimonial section
- Footer
- Responsive design (mobile & desktop)

## 🛠️ Teknologi
- HTML5
- CSS3
- Flexbox
- Google Fonts (Poppins)
